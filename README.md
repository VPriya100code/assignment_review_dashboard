# 🚀 Assignment & Review Dashboard

A modern, responsive **Assignment Management Dashboard** built using **React.js** and **Tailwind CSS**.
This application enables students to track assignments and confirm submissions, while admins (professors) can create and monitor assignments with real-time progress visualization.

---

## 📌 Features

### 👨‍🎓 Student

* View assigned tasks
* Track submission status (Pending / Submitted / Overdue)
* Double confirmation submission flow
* Progress tracking with visual indicators
* Deadline countdown & urgency indicators

### 👨‍🏫 Admin

* Create, edit, and delete assignments
* Attach external submission links (Google Drive, etc.)
* Monitor student submission status
* View progress bars for each student

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Language:** JavaScript / TypeScript
* **State Management:** React Hooks (useState, useEffect)
* **Data Storage:** localStorage (No backend)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/assignment-dashboard.git
```

2. Navigate to the project folder:

```
cd assignment-dashboard
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:5173/
```
